About ALON

02

ALON (Always On) is a personal AI assistant for the entire home-buying journey. Built by TryThat.ai, ALON guides buyers through all 8 stages of property acquisition — from initial search to final possession — acting exclusively in the buyer's interest.

Unlike traditional real estate platforms that serve listings, ALON serves buyers. It never shares your contact without permission, provides complete journey support (not just listings), and always works for you — never the builder or agent.

The design language reflects Trust Architecture — a visual system built on deep navy foundations and warm terracotta accents that convey institutional reliability with approachable warmth.

Icons · Animation · Rules

09
Iconography — Lucide React Native
All icons use the Lucide library (stroke-based). Default stroke width: 1.8. Use 2.0 for emphasis (close, checkmark).

13-14px Inline with text, secondary actions SlidersHorizontal, Mic, Pencil

18-20px Navigation, top bar actions ChevronLeft, ChevronRight, X
24px Alert cards, feature icons ShieldCheck, Route, UserCheck

Design Principles
Trust First
Every visual choice reinforces trust and reliability. Navy conveys institutional confidence, terracotta adds warmth.
Flat Over Skeuomorphic
Never heavy shadows. Use subtle elevation with navy-tinted shadows. On blue backgrounds, use border + bg opacity — no shadows.
One CTA Rule
One primary action per screen with solid fill. Secondary actions use outlined or ghost variants for clear hierarchy.
Active = Border + Fill
Selected states always show both border color change AND background fill change. Never just color alone.

React Native + Expo Expo Router Reanimated Zustand Lucide Icons Gesture Handler
